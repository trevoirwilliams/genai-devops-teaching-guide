# genai-devops-teaching-guide

This is the finished project for the lessons, including the infra/pipelines/docs built during the live demos.

## Prereqs
- .NET 10 SDK
- Git
- VS Code
- GitHub Copilot

## Build and test locally
```bash
dotnet restore
dotnet build
dotnet test
```
